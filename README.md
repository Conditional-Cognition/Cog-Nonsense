# Nonsense

A Minecraft mod for NeoForge 1.21.1 that adds tables, teacups, teabags, tablespoons, teaspoons, and tea buckets/bottles.

## Features
- Wood and stone tables
- Teacups, teabags, and tea items (black, green, oolong, chamomile)
- Tablespoons and teaspoons in various materials
- Funny Music Disc

## Requirements
- Minecraft 1.21.1
- NeoForge Modloader
