# Nonsense

Teaspoons and Tablespoons, in summary.

## History
- The mod was initially made as a joke and it continues to be so. The joke was 'If teaspoons spin tea then tablespoons spin tables.' likewise that's exactly what the mod does.

## Features (as of most recent version)
- Wood and stone tables
- Teacups, teabags, and tea items (black, green, oolong, chamomile)
- Tablespoons and teaspoons in various materials
- Tablespoons, when used on a Table, rotate it on the Y axis
- Teaspoons have no use currently (WIP)
- Funny Music Disc

## Requirements
- Minecraft 1.21.1
- NeoForge Modloader
